Carrossel Animado com Firebase

Uma solução completa de carrossel integrada com Firebase Firestore, pronta para hospedagem no GitHub Pages

Hospedagem no GitHub Pages com Firebase

1. Configurar o Firebase

Crie um projeto no Firebase Console e configure o Firestore:

Coleção: carousel

Documentos com campos: title, name, description, image, button1, button2, order

2. Criar Repositório

Crie um novo repositório no GitHub com seu projeto:

https://github.com/[seu-usuario]/[seu-repositorio]

Adicione os arquivos HTML, CSS e JS do carrossel

3. Configurar GitHub Pages

No repositório, vá em Settings > Pages:

Selecione o branch (geralmente main ou gh-pages) e a pasta (root ou /docs)

Salve e aguarde a publicação

4. Configurar Regras de Segurança

No Firebase, configure as regras do Firestore para permitir leitura pública:

rules_version = '2';
service cloud.firestore {
match /databases/{database}/documents {
match /carousel/{document=**} {
allow read: if true;
}
}
}
